# 💗 Apology Website

A romantic apology website built using HTML, CSS, and JavaScript. This project was created to express genuine feelings, take responsibility, and communicate emotions in a more meaningful and thoughtful way than a simple message.

## 🌸 Purpose

This website was made to express a sincere apology, show love, care, and emotional honesty, communicate feelings in a creative way, and ask for forgiveness with respect and sincerity.

## 💻 Features

Soft romantic pink aesthetic design, smooth animations and scroll effects, floating heart background animation, embedded Spotify music player, emotional apology section, “Reasons I Love You” cards, photo gallery for personal memories, interactive forgiveness buttons including Accept Apology and Not Yet, and a fully responsive design for both mobile and desktop.

## 🚀 How to Run

Download or clone the repository, make sure all files are in the same folder, and open index.html in your browser to view the project.

## 🌐 Deployment

This project can be hosted for free using GitHub Pages, Netlify, or Vercel.

This project is personal and emotional in nature. It was built with sincerity and care to communicate feelings that are difficult to express through text alone.
